# Task Manager

A simple web application built with React, Node.js, and SQLite for managing tasks. Each user can register, log in, and manage their own set of tasks, including creating, updating, viewing, and deleting tasks. This project demonstrates basic user authentication, task CRUD operations, and a clean, responsive UI.

## Features

- User authentication (registration, login, and logout)
- Task management (create, read, update, delete)
- Each user has their own set of tasks
- Responsive and clean user interface
- Secure password storage using bcrypt
- Token-based authentication using JWT

## Tech Stack

- **Frontend**: React
- **Backend**: Node.js, Express.js
- **Database**: SQLite
- **Authentication**: JWT (JSON Web Tokens), bcrypt for password hashing
- **Styling**: Custom CSS

## Prerequisites

Ensure you have the following installed:

- [Node.js](https://nodejs.org/) (v12 or above)
- [SQLite3](https://www.sqlite.org/) (for managing the database)
